Mario Clone (2019)

Project Info:

I developed this project for my Game Engines Creation module during my first year of university, programming it in C++ using the SDL library. The end result is a clone of the original arcade Mario Bros game, and a clone of the original Super Mario Bros for the NES, all implemented into one game. Features include one and two-player modes for both games, a functional level editor that works for both games simultaeniously, high scores for both games,and functional user interface components that are used in the main menu and level editor. Game-related features include working collision, AI, animation, sound, and gameplay functionality such as death, score, victory conditions (for Super Mario Bros) and collectibles (coins). I earned an overall grade of 97% for this assignment.

What went well?

  • Both games were implemented successfully and strongly resembled the source material
  • Making the level editor, main menu and high score tables taught me a lot about developing UI components from scratch

How would I improve it?

  • My collision blocking had minor unexpected behaviour in a few edge cases
    • With more time, I would have researched a more robust method of collision blocking
  • The level editor could have been more user-friendly
    • More hotkeys could have boosted workflow efficiency
  • My implementation for UI buttons could have been improved
    • The current system performs AABB checks on each button onscreen and then
